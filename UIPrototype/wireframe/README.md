# PaperMate 低保真线框图 v2

> 依据《页面设计.docx》重构，含完整页面跳转与交互逻辑

## 启动

```bash
cd prototype
python -m http.server 8080
# http://localhost:8080
```

或双击 `index.html`。

---

## 页面结构（6 页）

| 编号 | 页面 | 说明 |
|------|------|------|
| 页面一 | 登录/注册 | 账号登录、注册、首次引导 |
| 页面二 | 论文检索与发现（工作空间） | 推荐论文、搜索/提问、论文卡片 |
| 页面三 | 论文详情 | 左侧三 Tab + 右侧四栏侧边栏（全部/信息/问答/辅助阅读） |
| 页面四 | 学习空间 | 收藏/笔记/历史/词典/画像/模式切换 |
| 页面五 | 管理员后台（系统管理端） | 六模块：概览 / Agent舰队 / 任务队列 / 质量异常 / 配置治理 / 审计日志 |
| 页面六 | 设置 | 抓取订阅/账户/网页设置 |

---

## 左侧导航

**工作空间**、**学习空间**、**管理员后台 [管理员]**、**设置**

登录后默认进入 **页面二 · 工作空间**。

---

## 工作空间布局逻辑

**初始状态：**
- 显示「每日 ArXiv 精选列表」+「基于画像推荐的论文」
- **不显示**「论文卡片视图」（搜索结果列表）

**搜索 / 提问后：**
- 移除推荐论文区域
- 显示「论文卡片视图」，展示检索/抓取结果
- 支持多轮追问并更新结果列表

## 页面与交互流程

```
页面一(登录/注册)
  ├─ 登录 ──────────────────────→ 页面二
  └─ 注册 → 首次引导 ──────────→ 页面二

页面二(工作空间)
  ├─ 初始：每日精选 + 画像推荐论文卡片
  ├─ 搜索/提问：移除推荐 → 显示检索结果列表
  └─ 点击论文卡片 → 页面三

页面三(论文详情)
  ├─ 左侧：a 论文主体 / b 智能总结 / c 知识图谱
  └─ 右侧侧边栏：全部 / 信息 / 问答 / 辅助 / 笔记 / 对比
      · 全部：分区卡片展示，点击卡片跳转对应 Tab；问答发送后自动展开问答 Tab
      · 信息：论文元数据 + 知识 Wiki 检索（标题/作者/关键词/方向/概念）
      · 对比：双栏对比阅读，快捷切换 A/B 论文
      · 笔记：高亮段落 + 笔记列表 + 评论

页面四(学习空间)
  └─ Tab: 收藏/笔记/历史/词典/画像/模式切换

页面五(管理员后台)
  ├─ 系统概览：核心指标 / Agent健康 / 活动流 / 待办
  ├─ Agent舰队：列表运维 / 24h指标下钻
  ├─ 任务队列：看板 + 任务列表 / 处理日志
  ├─ 质量异常：异常论文工作台 / 人工修正 / 质量统计
  ├─ 配置治理：订阅规则 / 用户管理
  └─ 审计日志：操作审计 / 系统运行日志

页面六(设置)
  └─ Tab: 抓取订阅/账户/网页
```

## 交互清单

| 交互 | 位置 | 行为 |
|------|------|------|
| 登录/注册切换 | 页面一 | Tab 切换表单 |
| 首次引导弹窗 | 注册后 | 选方向+画像 → 页面二 |
| 推荐论文展示 | 页面二初始 | 每日精选 + 画像推荐 |
| 搜索/提问 | 页面二 | 移除推荐区，显示检索结果列表 |
| 论文卡片跳转 | 页面二/四 | → 页面三论文详情 |
| 阅读历史 | 页面四 | 点击日期展示缩略论文卡片 |
| 模式切换 | 页面四 · 默认模式 Tab | 设置全局默认阅读模式 |
| 辅助阅读 | 页面三 · 辅助阅读 | 按模式生成个性化总结，可快捷切换 |
| 对比阅读 | 页面三 · 对比 Tab | 双论文并排对比，下拉/快捷切换 A/B |
| 论文知识 Wiki | 页面三 · 信息 | 多维度检索关联论文，点击标签快捷搜索 |
| 笔记与评论 | 页面三 · 笔记 Tab | 高亮笔记、评论发表，全部页预览 |
| 管理员后台 | 页面五 | 六模块系统管理 · 账号 `admin` 登录 |
| 管理员登录 | 页面一 | 账号 `admin` 识别为管理员 |
