## 0. 四阶段一览（验收速查）

| Spike | 阶段 | 输入 | 预期输出 | 超时 | 失败判定 / 状态 |
|-------|------|------|----------|------|-----------------|
| S1 | 抓取 Fetch | arXiv ID（可选 PDF/HTML） | 元数据 + PDF 路径；`fetched` | 元数据 ≤10s；PDF ≤60s（长文 ≤120s） | HTTP≥400 / 空文件 / 超时 / 无效 ID → `fetch_failed` |
| S2 | 解析 Parse | `fetched` 的 PDF（P7 可对照 HTML） | 章节/段落块；公式占位；表文本；`parsed` 或 `parsed_degraded` | 正常 ≤90s；P5/P8 ≤180s | 无文本 / 严重串栏 / 应有公式表却 0 → `parse_failed` / `parsed_degraded`；超时 → `parse_timeout` |
| S3 | 摘要 Summarize | `parsed`/`parsed_degraded` + 元数据 | `summary` / `concept` / `methods`；`summarized` | ≤60s；长文 ≤120s | 关键字段空 / 领域无关 / 虚构表内数字 / 超时 → `summarize_failed` |
| S4 | 问答 QA/RAG | 论文 ID + 固定题（≥2） | 回答 + ≥1 可核验引用；对齐 UC-P03 | RAG ≤3s；端到端 ≤30s | 超时 / 无出处答细节 / 出处矛盾 / 失败样例未拒 → `qa_failed` / `qa_timeout` |

### 流水线状态（约定）

```
pending → fetching → fetched → parsing → parsed
       → summarizing → summarized → qa_ready

旁路：fetch_failed | parse_timeout | parse_failed | parsed_degraded
     | summarize_failed | qa_timeout | qa_failed
```

**硬约束**：P9/P10 一旦 `fetch_failed`，不得进入解析/摘要；问答必须拒绝并**禁止假引用**。

## Spike-S1：抓取（Fetch）

| 项 | 定义 |
|----|------|
| **输入** | arXiv ID；可选同时拉元数据 + PDF（P7 可加 HTML/ar5iv） |
| **预期输出** | 元数据（title / authors / abstract / categories）+ PDF 存储路径；状态 = `fetched` |
| **超时** | 元数据 API ≤ **10s**；PDF 下载 ≤ **60s**；长文 P5/P8 可放宽至 **120s** |
| **失败判定** | HTTP ≥ 400；0 字节或非 PDF；超时；无效 ID → `fetch_failed` |

| 样例 | 期望状态 |
|------|----------|
| P1–P8 | `fetched` |
| P9、P10 | `fetch_failed`（错误码可读，如 404） |

---

## Spike-S2：解析（Parse）

| 项 | 定义 |
|----|------|
| **输入** | `fetched` 的 PDF（P7 可对照 HTML）；标签：双栏 / 公式 / 长文 |
| **预期输出** | 章节或段落块；公式占位/LaTeX；表格文本或结构；`parsed` 或 `parsed_degraded` |
| **超时** | 正常 ≤ **90s**；P5/P8 ≤ **180s**；超时 → `parse_timeout` |
| **失败判定** | 无文本；双栏严重串栏；公式/表格预期密集却检出 0 → `parse_failed` / `parsed_degraded` |

| 样例 | 关注点 |
|------|--------|
| P1 | 基线可读 |
| P2、P6 | 双栏不错列 |
| P3 | 公式段非空 |
| P4 | 实验表有文本 |
| P5、P8 | 不卡死，可有截断策略 |
| P9、P10 | **不进入**解析 |

---

## Spike-S3：摘要（Summarize）

| 项 | 定义 |
|----|------|
| **输入** | `parsed` / `parsed_degraded` 块 + 元数据 |
| **预期输出** | `summary` / `concept` / `methods`；状态 = `summarized` |
| **超时** | ≤ **60s**；长文 ≤ **120s** |
| **失败判定** | 关键字段空；领域无关；虚构表内数字；超时 → `summarize_failed` |

| 样例 | 期望 |
|------|------|
| P1、P2、P4、P7 | 三件套非空 |
| P3 | 要点贴合正文（不必抄全公式） |
| P5、P8 | 可压缩，不得空 |
| P9、P10 | **不得**装成成功摘要 |

## Spike-S4：问答（QA / RAG）

| 项 | 定义 |
|----|------|
| **输入** | 论文 ID + 固定题（每篇 ≥2：事实 + 方法） |
| **预期输出** | 回答 + ≥1 可核验引用（chunk_id / section / 页码）；对齐 UC-P03 POST-S2 / POST-F1 |
| **超时** | RAG ≤ **3s**；含生成端到端 ≤ **30s** |
| **失败判定** | 超时；无出处答细节；出处矛盾；失败样例未拒绝 → `qa_failed` / `qa_timeout` |
