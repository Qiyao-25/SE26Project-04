# 技术 Spike 清单

> 覆盖 [10篇样例论文清单.md](./10篇样例论文清单.md)。每阶段统一四列：**输入 | 预期输出 | 超时 | 失败判定**。

## 0. 流水线状态（约定）

```
pending → fetching → fetched → parsing → parsed
       → summarizing → summarized → qa_ready

旁路：fetch_failed | parse_timeout | parse_failed | parsed_degraded
     | summarize_failed | qa_timeout | qa_failed
```

---

## Spike-S1：抓取（Fetch）

| 项 | 定义 |
|----|------|
| **输入** | arXiv ID；可选同时拉元数据 + PDF（P7 可加 HTML） |
| **预期输出** | 元数据（title / authors / abstract / categories）+ PDF 存储路径；状态 = `fetched` |
| **超时** | 元数据 API ≤ **10s**；PDF 下载 ≤ **60s**；长文 P5/P8 可放宽至 **120s** |
| **失败判定** | HTTP ≥ 400；0 字节或非 PDF；超时；无效 ID → `fetch_failed` |

| 样例 | 期望状态 |
|------|----------|
| P1–P8 | `fetched` |
| P9、P10 | `fetch_failed`（错误码可读） |

---

## Spike-S2：解析（Parse）

| 项 | 定义 |
|----|------|
| **输入** | `fetched` 的 PDF（P7 可对照 HTML）；标签：双栏 / 公式 / 长文 |
| **预期输出** | 章节或段落块；公式占位/LaTeX；表格文本或结构；`parsed` 或 `parsed_degraded` |
| **超时** | 正常 ≤ **90s**；P5/P8 ≤ **180s**；超时 → `parse_timeout` |
| **失败判定** | 无文本；双栏严重串栏；公式/表格预期密集却检出 0 → `parse_failed` / `parsed_degraded` |

| 样例 | 关注点 |
|------|--------|
| P1 | 基线可读 |
| P2、P6 | 双栏不错列 |
| P3 | 公式段非空 |
| P4 | 实验表有文本 |
| P5、P8 | 不卡死，可有截断策略 |
| P9、P10 | 不进入解析 |

---

## Spike-S3：摘要（Summarize）

| 项 | 定义 |
|----|------|
| **输入** | `parsed` / `parsed_degraded` 块 + 元数据 |
| **预期输出** | `summary` / `concept` / `methods`；状态 = `summarized` |
| **超时** | ≤ **60s**；长文 ≤ **120s** |
| **失败判定** | 关键字段空；领域无关；虚构表内数字；超时 → `summarize_failed` |

| 样例 | 期望 |
|------|------|
| P1、P2、P4、P7 | 三件套非空 |
| P3 | 要点贴合正文（不必抄全公式） |
| P5、P8 | 可压缩，不得空 |
| P9、P10 | **不得**装成成功摘要 |

---

## Spike-S4：问答（QA / RAG）

| 项 | 定义 |
|----|------|
| **输入** | 论文 ID + 固定题（每篇 ≥2：事实 + 方法） |
| **预期输出** | 回答 + ≥1 可核验引用；对齐 UC-P03 |
| **超时** | RAG ≤ **3s**；含生成端到端 ≤ **30s** |
| **失败判定** | 超时；无出处答细节；出处矛盾；失败样例未拒绝 → `qa_failed` |

### 固定测试题

| 样例 | 题1（事实） | 题2（方法） |
|------|-------------|-------------|
| P1 | Transformer 是否依赖 CNN/RNN？ | Multi-Head Attention 的作用？ |
| P2 | BERT 预训练任务有哪些？ | 双向编码器如何体现？ |
| P3 | Neural ODE 把网络深度看成什么？ | 训练时如何反向传播？ |
| P4 | LoRA 更新什么矩阵？ | 相对全量微调参数量如何？ |
| P5 | Survey 主要覆盖哪类模型？ | 文中主题如何划分？（概括） |
| P6 | ViT 如何切分图像？ | 与 CNN 归纳偏置差异？ |
| P7 | RAG 结合哪两类能力？ | 检索器与生成器如何协作？ |
| P8 | GPT-3 参数量级？ | few-shot 指什么？ |
| P9/P10 | 任意题 | 须提示不可用，**禁止假引用** |
